
#Rohit's Dance Academy

This Is A multi page web Apppliaction that stores information from contact page in mongoDB.

## Tech Stack

**Client:** Pug & Css

**Server:** Node, Express

**Storage:** MongoDB



## Run Locally

Clone the project

```bash
  git clone git@github.com:Rohitpanti/Dance-Academy.git
```


Install dependencies

```bash
  npm install
```

Start the server

```bash
  nodemon server.js
```
